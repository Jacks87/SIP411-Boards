Program Objectives

Objective 1 - Develop, analyze and integrate artificial intelligence applications and IoT systems.

Mystic Card Forge

This project demonstrates integration of AI applications by combining a web interface, Firebase persistence, and the AI card generator, simulating workflows similar to IoT systems that manage data across devices.

HyperStopwatch Mini

This project applies AI to real-world productivity tracking by integrating baseline vs AI-assisted timing, showing how AI systems can be analyzed and integrated for performance improvements.

Objective 2 - Design and create natural language processing systems (NLP).

Bargenius

This project processes live call transcripts and provides AI-driven coaching, demonstrating practical application of NLP techniques for real-time communication.

FeedbackLens Mini

This tool analyzes written feedback using AI sentiment and keyword extraction, showing design and creation of NLP-based systems for improving user experience.

Objective 3 - Design and create machine learning systems using best practices and patterns.

AI Music Generator

By training models on Spanish pop/reggaeton datasets, this project applies machine learning patterns to generate new audio content, following best practices in preprocessing and evaluation.

Movie Sorter

This web app demonstrates fundamental ML concepts like sorting and data management, while adhering to best practices in user input validation and algorithm design.

Objective 4 - Innovate in deep learning by consuming big data with original algorithms.

Ritmo AI Studios

Generates AI music across genres by consuming large datasets of audio and applying deep learning models, producing original reggaeton/pop tracks that highlight algorithmic innovation.

Abstract Illusion Museum

Uses deep learning tools to generate and curate large sets of AI-created abstract images, demonstrating innovation in creative algorithms that process big visual datasets.

Objective 5 - Demonstrate software development skills using multiple languages, environments, platforms, and source control.

Checkers Game

Built in JavaScript, this project highlights programming fundamentals while reinforcing skills in logic, game loops, and browser environments.

Mystic Card Forge

Developed across multiple platforms (VS Code, GitHub Pages, Firebase), this project demonstrates cross-environment development and source control best practices.

Objective 6 - Describe, develop, analyze, and integrate data structures, databases, and database management systems.

Mystic Card Forge

Card data and images are structured, stored, and retrieved with Firebase + localStorage, showing database integration and schema-like organization of user inputs.

HyperStopwatch Mini

Stores session data for analysis and comparison, demonstrating database management techniques and structured integration of performance results.
